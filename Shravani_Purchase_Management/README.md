# 🛒 Purchase Management System

## 📌 Overview
This is a web-based Purchase Management System developed using DBMS concepts. It helps manage suppliers, products, and purchase transactions efficiently.

## 🚀 Features
- Add, view, update, and delete suppliers
- Manage product inventory
- Record purchase transactions
- Display data in tabular format
- Generate realistic sample data

## 🛠️ Technologies Used
- Python
- Streamlit
- SQLite (Database)

## 🧩 Database Tables
- Supplier
- Product
- Purchase
