# Attendence-management-system

A full-stack Attendance Management System built with React and Node.js/Express.

## Features
- Employee CRUD operations
- Real-time attendance tracking
- Leave request workflows
- Notification system
- Admin and Employee roles
- Premium dark-mode UI

## Tech Stack
- **Frontend:** React, Vite
- **Backend:** Node.js, Express
- **Database:** SQLite

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
